# Rock Paper Scissors (Console JS)

A simple Rock Paper Scissors game built in JavaScript.  
Runs entirely in the browser using `prompt()` and `console.log()`.

The player competes against a computer over 5 rounds.

## 👥 Team Project

This website was created as a team project by me and my teammates.

## 📌 How It Works

- The computer generates a random choice (`rock`, `paper`, `scissors`)
- The player puts their choice with a `prompt()`
- Invalid inputs repeat the round
- Scores are tracked
- Winner is displayed at the end


## 🚀 Game Rules

- Rock beats Scissors  
- Scissors beats Paper  
- Paper beats Rock
